# NextRides.com - Требования и Архитектура

## Оригинальное ТЗ
Создать веб сайт - каталог объявлений о продаже автомобилей NextRides.com.
- Пользователи создают аккаунт и размещают объявления
- Обязательные поля: минимум 3 фото, марка, модель, год, пробег, цена, тип привода, город, zip код, телефон, VIN, описание (мин. 10 символов)
- Шапка: овальный логотип, кнопка Catalog, зелёная кнопка Sell Car / My Listings
- Быстрый поиск: Make, Model, Zip Code, Distance, Show Matches

## Реализованный функционал

### Backend (FastAPI + MongoDB)
- **Аутентификация**: JWT-based (регистрация, вход, профиль)
- **Объявления CRUD**: создание, чтение, редактирование, удаление
- **Загрузка изображений**: локальное хранение в /uploads
- **Поиск и фильтры**: по марке, модели, году, пробегу, цене, типу привода, zip коду

### Frontend (React + Tailwind CSS)
- **Главная страница**: быстрый поиск, сетка объявлений
- **Шапка**: sticky header с логотипом, Catalog, Sell Car/My Listings
- **Модал авторизации**: регистрация/вход
- **Модал фильтров**: расширенный поиск
- **Страница объявления**: галерея фото, все данные, контакт продавца
- **Личный кабинет**: My Listings с редактированием/удалением
- **Создание объявления**: форма со всеми обязательными полями

## Структура файлов
```
/app/backend/
  - server.py          # API endpoints
  - .env               # Конфигурация
  - uploads/           # Хранение изображений

/app/frontend/src/
  - App.js             # Роутинг и контекст
  - components/
    - Header.jsx       # Шапка
    - AuthModal.jsx    # Авторизация
    - FiltersModal.jsx # Фильтры
    - QuickSearch.jsx  # Быстрый поиск
    - CarCard.jsx      # Карточка авто
  - pages/
    - HomePage.jsx        # Главная
    - CarDetailPage.jsx   # Детали объявления
    - MyListingsPage.jsx  # Личный кабинет
    - CreateListingPage.jsx # Создание/редактирование
```
